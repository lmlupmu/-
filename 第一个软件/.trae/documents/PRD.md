# 产品需求文档（PRD）

## 1. 产品概述

声音音色检测网页是一款面向普通用户的趣味语音分析工具。用户通过选择性别、朗读系统随机提供的古诗，即可获得基于"大数据音色库"的音色类型判定与音乐风格推荐。

- 主要目的：以低门槛的语音互动方式，帮助用户了解自己的声音特质，并提供娱乐化的音色分析报告。
- 目标用户：对自我声音特征好奇、喜欢社交分享的年轻用户群体。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，打开网页即可使用 | 选择性别、录音、查看音色分析报告 |

### 2.2 功能模块

1. **性别选择页**：蓝色"男生"按钮与粉色"女生"按钮，选择后进入录音页。
2. **古诗朗读与录音页**：随机展示一首古诗，提供 QQ 风格的按住说话按钮，实时显示录音时长。
3. **音色分析结果页**：展示音色类型、判断理由、音色稀有度、适合演唱的音乐类型等简介。

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 性别选择页 | 性别选择区 | 展示标题与两个大按钮，男生蓝色、女生粉色，点击后记录性别并跳转 |
| 古诗朗读页 | 古诗展示区 | 从预设古诗库中随机抽取一首展示，提供换一首功能 |
| 古诗朗读页 | 录音控制区 | QQ 风格圆形语音按钮，按住开始录音，松开结束，实时显示录音秒数 |
| 分析结果页 | 结果卡片区 | 展示音色类型、稀有度星级、判断理由、适合音乐类型、形象标签 |

## 3. 核心流程

用户进入网页后，首先在性别选择页点击"男生"或"女生"按钮；系统记录性别后进入古诗朗读页，并随机展示一首古诗。用户按住语音按钮朗读古诗，页面实时显示录音时长；松开按钮后，系统基于录音音频特征与内置音色大数据模型进行匹配分析，最终跳转至音色分析结果页，展示完整的音色报告。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色背景（`#0f172a`）搭配渐变光晕，营造科技感与神秘感。
- **性别按钮**：男生使用蓝色渐变（`#3b82f6` → `#1d4ed8`），女生使用粉色渐变（`#ec4899` → `#be185d`），按钮为圆角大卡片，带有悬停放大与微光效果。
- **字体**：标题使用具有力量感的现代无衬线字体（如 `Noto Sans SC` 加粗），正文使用清晰易读的黑体。
- **布局**：居中卡片式布局， generous 留白，页面切换使用淡入淡出过渡动画。
- **语音按钮**：QQ 风格大圆形按钮，按住时显示脉冲动画与红色录音状态，周围环绕动态声波动画。
- **结果页**：采用数据仪表盘风格，音色类型大字号突出，稀有度以星级展示，整体使用玻璃拟态卡片。

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 性别选择页 | 标题区 | 居中大字标题"发现你的声音音色"，副标题引导语 |
| 性别选择页 | 性别按钮区 | 两个大按钮横向排列，蓝色男生、粉色女生，带图标与悬停动效 |
| 古诗朗读页 | 顶部导航 | 返回按钮、当前步骤提示 |
| 古诗朗读页 | 古诗卡片 | 玻璃拟态卡片展示诗名、作者、诗句，提供"换一首"按钮 |
| 古诗朗读页 | 录音区 | QQ 风格语音按钮、录音时长、提示文字 |
| 分析结果页 | 结果标题 | "你的音色是"引导至音色类型大标题 |
| 分析结果页 | 稀有度 | 星星评分，搭配稀有度文案 |
| 分析结果页 | 分析详情 | 判断理由、适合音乐类型、形象标签等卡片 |

### 4.3 响应式设计

- 桌面优先，最大内容宽度 480px，居中显示，适配大屏与小屏。
- 性别按钮在桌面端横向排列，在窄屏自动堆叠为纵向。
- 语音按钮大小随屏幕宽度自适应，确保移动端可轻松按住操作。
- 使用 `viewport` 适配移动设备，禁止页面缩放，保证 App 式体验。

### 5.3 音色判定维度

系统从以下维度对录音进行特征提取与匹配：

1. **基频（Pitch）**：判断声音高低，区分低音、中音、高音类型。
2. **语速（Speaking Rate）**：统计单位时间内字词数量，判断快慢节奏。
3. **能量/响度（Energy）**：分析音频 RMS 能量，判断声音厚重或清亮。
4. **频谱重心（Spectral Centroid）**：区分明亮音色与温暖音色。
5. **泛音丰富度（Harmonic Complexity）**：判断沙哑感、磁性或纯净感。
6. **时域抖动（Jitter）**：评估声音稳定性，辅助判断气泡音等特征。
